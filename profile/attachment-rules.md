# Attachment Rules

## Hard rules
- Do not auto-attach sensitive internal reports or screenshots
- Assets marked `requiresManualReview=true` must never be auto-attached
- If `safeToAttach=false`, mention-only is allowed when `safeToMention=true`
- If `safeToAttach=needs_review`, keep the asset out of auto-attach and surface it as recommend-only
- If the role is broad or weakly matched, prefer mention-only proof over attachments

## Good auto-attach cases
- Truly Beauty for beauty/skincare/Klaviyo roles
- Lifely for high-AOV / furniture / home goods / premium DTC roles
- Fly Boutique for fashion/apparel or deliverability rescue roles
- Design Case Studies for design/Figma/email design roles
- Endurance Wellness when the brief is about lifecycle foundation or subscription setup

## Mention-only defaults
- Dr. Rachael internal report
- DTC performance PDF when the exact client/brand match is uncertain
- Screenshots with internal-looking dashboard data
- Video/Figma links unless the client clearly asks for them

## Browser prep note
Attachment selection should prepare metadata for browser draft preparation only. It must not trigger live submit actions.
